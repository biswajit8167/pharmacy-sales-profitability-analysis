# Pharmacy Sales & Profitability Analysis Dashboard

## Project Overview

This project analyzes pharmacy sales, profitability, promotions, regional performance, and product analytics using Power BI.

The dashboard helps identify:

- Revenue trends
- Profitability insights
- High-performing pharmacies
- Product contribution
- Promotion effectiveness
- Geographic sales patterns
- Regional performance

# Key Business Insights

- Revenue increased significantly across major pharmacy regions.
- Prescription products contribute the highest profitability.
- Promotions improve sales volume but reduce margin percentage.
- Urban pharmacies outperform rural pharmacies in total revenue.
- Germany is the strongest revenue-generating country.
